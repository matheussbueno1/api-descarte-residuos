# api-descarte-residuos
Desenvolver uma API com NestJS (obrigatório) para registrar e consultar dados sobre o descarte de resíduos em uma determinada localidade, contribuindo para a conscientização ambiental e o uso responsável dos espaços públicos. A atividade se alinha ao Objetivo de Desenvolvimento Sustentável (ODS) 12 da ONU: "Consumo e Produção Responsáveis.
